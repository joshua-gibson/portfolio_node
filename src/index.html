The Adventurer's Shop

The project was time limited to 5 days. The purpose was to develop an inventory management system. I themed this after an 8-bit adventure RPG video game. The tech stack here is Ruby with Sinatra and PostgreSQL.

Rocket Launch Timeline

The project produced an educational app displaying an interactive timeline of rocket launches, from the 1960's going into the near future. For this we used the public Rocket Launches API. The project was built with Node.js in the back end and Vue in the front end. We also used MongoDB to cache the API data and make the queries quicker. We only had 5 days to produce a working MVP.

Dinosaur Park Manager

This was an ambitious project to produce a theme park management application. The back-end was Java with Spring, SpringBoot, and Hibernate, whiched produced a full RESTful API that managed a PostgreSQL database. The front end which interacted with the API was built with React. This was done over the course of 5 days.
